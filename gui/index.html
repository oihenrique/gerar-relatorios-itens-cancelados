Guia rápido

  1. Copie os dados do sistema
  2. Clique no botão para gerar a planilha geral
  3. Abra a planilha geral dentro da pasta "data" deste programa e faça uma filtragem dos dados relevantes
  4. Clique no botão para gerar planilhas por loja, elas são geradas dentro da pasta "data > store_sheets" caso queira conferir
  5. Clique para enviar as planilhas, elas serão enviadas para os e-mails contidos em: app > config > lista_emails_lojas
